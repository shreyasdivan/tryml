About TryML

View classifier performance
Run dataset through various machine learning classifiers provided by scikit-learn and analyse
View co-relation matrix
View the attribute behaviour via the co-relation matrix table and gauge and analyse linear relationships if any
Introduce Beginners
Being a beginner myself, it can be fascinating/encouraging to view the outcome of passing the dataset through basic implementations of scikit-learn, without having to code!

Workflow

Preprocessing
After the user provides the training data (.csv file), TryML does a bit of basic preprocessing. It imputes any missing values and performs the StandardScaler transformation.
Co-relation matrix
Using Pandas' 'corr' function, the co-relation matrix is obtained. In practice, developers use heatmaps for better visualization.
Classfier Analysis
TryML then trains the various scikit-learn classifiers with the training data. It then calculates the score for each classifier using scikit-learn's cross validation.

Please note

TryML will be functional only for supervised classification datasets. TryML runs the default implementations for the classifiers and preprocessing of input data provided by the scikit-learn library. The is just a basic utility to play around.
